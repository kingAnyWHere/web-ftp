# web-ftp
一款功能强悍的web磁盘管理工具  （A powerful web disk management tools）

# 快捷操作

- alt + o 打开文件夹
- alt + n 新建文件夹
- alt + m 配置web-ftp
- alt + w / alt + x 关闭文件夹
- alt + d 选中该路径

- ctrl + a 全选
- ctrl + c 复制
- ctrl + x 剪切
- ctrl + v 粘贴 (也可粘贴截图)

- ctrl + z / backspace  历史后退
- ctrl + y  历史前进

- ctrl + alt + f 自动排版
- delete 删除
- f2 重命名
- enter 打开
